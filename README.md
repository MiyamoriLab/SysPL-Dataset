# 「GPT-2 における体系性の分析」における命題論理データセット


## 内容物<a name="content"></a>[](#content)

- [01\_dataset](#01)

### 01_dataset
本研究において使用した命題論理データセット．実験で使用した学習データとテストデータが格納されている．

### 02_scr
命題論理データセット構築のプログラムファイル．

### 基本的なコマンド

```
python src/dataset_creator.py --n <データ数> --depth <論理式の深さ> --ops <演算子リスト>
```

### 引数 (Arguments)

| 引数 | 説明 | 型 | デフォルト値 |
| :--- | :--- | :--- | :--- |
| `--n` | 生成するデータサンプル数。 | `int` | `1000` |
| `--depth` | 生成する論理式の最大の深さ。値が大きいほど複雑な式が生成されます。 | `int` | `4` |
| `--ops` | 使用する論理演算子。`and`, `or`, `imply`, `not` から複数選択し、スペース区切りで指定します。 | `str` | `and or` |

## 使用例
```
python dataset_creater.py --n 200 --depth 4 --ops and or
```
