# 「GPT-2 における体系性の分析」における命題論理データセット

### 1. dataset
本研究において使用した命題論理データセット．実験で使用した学習データとテストデータが格納されている．

### 2. scr
命題論理データセット構築のプログラムファイル．

### 基本的なコマンド

```
python src/dataset_creator.py --n <データ数> --depth <論理式の深さ> --ops <演算子リスト>
```

### 引数 (Arguments)

| 引数 | 説明 | 型 | デフォルト値 |
| :--- | :--- | :--- | :--- |
| `--n` | 生成するデータサンプル数． | `int` | `1000` |
| `--depth` | 生成する論理式の最大の深さ． | `int` | `4` |
| `--ops` | 使用する論理演算子．`and`, `or`, `imply`, `not` から選択し，スペース区切りで指定． | `str` | `and or` |

## 使用例
```
python src/dataset_creater.py --n 200 --depth 4 --ops and or
```
